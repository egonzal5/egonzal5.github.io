  \frame{
    \frametitle{MCAR}

      The probability of having a `1` in a column is not dependent on
      \blueit{any} of the data, observed or unobserved.

      \onslide<2>{You can test this using conditional probability for the observed data.}
      \onslide<3>{But how realistic is this?}

  }

  \frame{
    \frametitle{MCAR: Not very realistic}

      When it comes to data we've collected, especially from \blueit{people}, completely
      random mechanisms are very unlikely:

      \begin{enumerate}
        \item<2 -> Older folks less like to answer questions online
        \item<3 -> In long-term studies of non-trivial sample sizes, people will die before study is complete
        \item<4 -> People/Institutions are often reluctant to reveal financial information
      \end{enumerate}
  }

  \frame{
    \frametitle{MAR}

      The probability of having a `1` in a column \blueit{is} dependent on
      \blueit{observed} data.

      \onslide<2>{This allows us to model the mechanism for when the data is missing.}
      \onslide<3>{We use \blueit{observed} data as input into our model.}

  }

  \frame{
    \frametitle{MAR: Pretty realistic}

      Can you think of examples?

      \onslide<2>{Because we can model for it, we can \blueit{compensate} for it!}

  }

  \frame{
    \frametitle{MNAR}

      The probability of having a `1` in a column \blueit{is} dependent on
      \blueit{unobserved} data.

      \onslide<2>{You can't ignore this.}
      \onslide<3>{Any analysis including MNAR data must model and guess what the missing data is, otherwise what's the point?}

  }
